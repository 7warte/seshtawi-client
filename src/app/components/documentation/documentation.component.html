Create a new article

Article structure

New article, HTTP payload 

            {{this.new_article | json}}

This API endpoint facilitates the creation of a new blog article within the system. To create an article, the request payload must adhere strictly to the structure defined by the BlogArticle interface. Each article must contain all required components, ensuring completeness and accuracy in the article's content.

Attributes (Required):

  • timestamp: Represents the date and time of the article's creation.
  • main_title: Refers to the primary title of the article.
  • subtitle_1 and body_1: Essential components comprising the first section of the article, providing a subtitle and corresponding content.
  • caption: Represents a brief description or caption associated with the article.
  • tags: An array of strings containing keywords or labels to categorize and identify the article.

Usage Notes:

  • Ensure that the timestamp field reflects the precise creation date and time.
  • The main_title field should encapsulate the core topic of the article.
  • Both subtitle_1 and body_1 sections must be provided to create the initial content of the article.
  • The caption field offers a concise summary or teaser for the article's content.
  • Utilize the tags array to associate relevant keywords or categories with the article for easy classification and retrieval.
